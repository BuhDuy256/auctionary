# Auctionary - Hướng dẫn Chạy Dự án

**Công nghệ**:
- Backend: **Express.ts**
- Frontend: **React.ts**

## Cài đặt & Thiết lập
1.  **Backend**: `cd backend` -> `npm install` -> Sao chép nội dung được cung cấp vào `.env`.
2.  **Frontend**: `cd frontend` -> `npm install` -> Sao chép nội dung được cung cấp vào `.env`.

## Cách chạy

### Windows (Chạy nhanh)
Chạy `./run.bat` tại thư mục gốc. Script này sẽ khởi động cả backend và frontend.

### Chạy Thủ công
- Backend: `cd backend` -> `npm run dev`
- Frontend: `cd frontend` -> `npm run dev`
