# Auctionary - Project Guide

**Tech Stack**:
- Backend: **Express.ts**
- Frontend: **React.ts**

## Setup & installation
1.  **Backend**: `cd backend` -> `npm install` -> Copy provided keys to `.env`.
2.  **Frontend**: `cd frontend` -> `npm install` -> Copy provided keys to `.env`.

## How to Run

### Windows (Quick Start)
Run `./run.bat` in the root folder. This starts both backend and frontend.

### Manual Start
- Backend: `cd backend` -> `npm run dev`
- Frontend: `cd frontend` -> `npm run dev`
